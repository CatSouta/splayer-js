# 参数

| 名称         | 默认值     | 描述                     |
| ------------ | ---------- | ------------------------ |
| container    | -          | 播放器容器元素           |
| autoplay     | false      | 音频自动播放             |
| loop         | false      | 音频循环播放             |
| volume       | 0.7        | 默认音量                 |
| audio        | -          | 音频信息，应该是一个对象 |
| audio.name   | -          | 音频名称                 |
| audio.artist | -          | 音频艺术家               |
| audio.url    | -          | 音频链接                 |
| audio.cover  | -          | 音频封面                 |
